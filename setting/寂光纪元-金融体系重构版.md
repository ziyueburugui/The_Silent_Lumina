# 《寂光纪元》金融体系完整设定（重构版）

**核心原则**：
1. **严格的收支平衡**：每个阶层的收入与支出必须合理匹配
2. **物价体系统一**：从面包到房产，所有物价相互关联
3. **印记者分层**：第1-2道印记者仍需为生计挣扎，第3道+才开始超脱
4. **材料为王**：高阶印记者的目标不是钱，而是稀缺材料
5. **社会恐惧**：多道印记者被警惕，影响他们的经济地位
