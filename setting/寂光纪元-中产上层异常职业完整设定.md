## **⟦ Ⅰ. 中产阶层职业（月入150-800币）⟧**

### **1. 医生（Doctor）**

**基本信息**：
- **年龄**：25-60岁
- **月收入**：400-800币
- **工作地点**：私人诊所、医院、富人家
- **学历要求**：医学院毕业（6年，总学费6000币）

---

#### **职业细分**

**A. 普通医生（General Practitioner）**
- **月收入**：400-500币
- **工作**：治疗常见疾病、简单手术
- **病人**：工人阶级、中产阶级
- **诊所**：租用，月租50-80币

**B. 外科医生（Surgeon）**
- **月收入**：600-800币
- **工作**：复杂手术（截肢、开颅、剖腹）
- **病人**：中产、上层
- **地点**：大医院、私人诊所

**C. 富人家庭医生（Private Physician）**
- **月收入**：800-1500币
- **工作**：专属服务一个富人家庭
- **病人**：贵族、大商人
- **待遇**：包吃包住，社会地位高

---

#### **工作内容**

**典型一天（普通医生）**：
```
07:00 - 起床
07:30 - 早餐（白面包+鸡蛋+牛奶+咖啡，50分）
08:00 - 到诊所
08:30-12:00 - 看诊（10-15个病人）
  - 工人的肺病（诊费10币+药费20币）
  - 中产的胃病（诊费20币+药费30币）
  - 孩子的发烧（诊费15币+药费10币）
12:00-13:00 - 午餐（餐馆，10币）
13:00-18:00 - 继续看诊（10-15个病人）
18:00-19:00 - 出诊（富人家，诊费50币）
19:00 - 回家
19:30 - 晚餐（家里，妻子做饭）
20:00 - 阅读医学期刊
22:00 - 睡觉

收入：
- 诊费：25病人 × 平均15币 = 375币
- 药费提成：10%
- 出诊：2次/周 × 50币 × 4周 = 400币
- 月收入：约500币
```

---

#### **职业压力**

**医疗事故**：
- 病人死亡 → 家属起诉 → 赔偿500-5000币
- 误诊 → 名誉受损 → 病人流失
- 手术失败 → 可能被吊销执照

**案例**：
```
【约翰医生的噩梦】

约翰，38岁，外科医生。

今天，他做了一个截肢手术。
病人是工厂工人，手被机器夹断。

手术过程：
- 麻醉：用乙醚（50币）
- 截肢：锯掉手臂
- 止血：烧灼伤口
- 缝合

但，感染了。

3天后，病人高烧不退。
5天后，病人死了。

家属冲进诊所：
"你这个庸医！杀人犯！"

约翰被起诉。

法院判决：
- 赔偿2000币
- 停业3个月

约翰破产了。

他花了10年攒的钱，
一次事故，全没了。

他现在欠债500币。

他还能干什么？

继续当医生。
因为这是他唯一会的。

但每次手术，他都在颤抖。
怕再次失败。
```

---

#### **社会地位**

**受尊敬**：
- 救死扶伤
- 学识渊博
- 体面职业

**但也被厌恶**：
- 收费贵（穷人看不起）
- 治不好病（医学落后）
- 有时被认为是"死神"

**与印记者的关系**：
- 普通医生：不知道印记者存在
- 少数医生：知道，但不敢说
- 守夜人医疗部医生：专门治疗印记者

### **2. 工程师/技师（Engineer/Technician）**

**基本信息**：
- **年龄**：25-55岁
- **月收入**：300-700币
- **工作地点**：工厂、建筑工地、设计院
- **学历要求**：技术学院毕业（4年，总学费3000币）

---

#### **职业细分**

**A. 蒸汽机工程师**
- **月收入**：500-700币
- **工作**：设计、维护大型蒸汽机
- **雇主**：工厂、铁路公司、船厂

**B. 建筑工程师**
- **月收入**：400-600币
- **工作**：设计建筑、监督施工
- **雇主**：建筑公司、政府

**C. 机械技师**
- **月收入**：300-500币
- **工作**：修理精密机械、制造零件
- **雇主**：工厂、修理铺

---

#### **工作内容**

**典型一天（蒸汽机工程师）**：
```
06:00 - 起床
06:30 - 早餐（全麦面包+咖啡，30分）
07:00 - 到工厂
07:30-12:00 - 检查蒸汽机
  - 测量压力表
  - 检查阀门
  - 听声音判断故障
  - 记录数据
12:00-12:30 - 午餐（工厂食堂，50分）
12:30-17:00 - 维修/设计
  - 更换磨损部件
  - 画设计图
  - 计算效率
17:00-18:00 - 写报告
18:00 - 下班
18:30 - 晚餐（家里）
19:00 - 阅读技术书籍
21:00 - 睡觉

月收入：600币（固定工资）
```

---

#### **职业风险**

**技术淘汰**：
- 新技术出现 → 旧技术工程师失业
- 必须不断学习

**工厂事故**：
- 蒸汽机爆炸 → 工程师被认为是责任人
- 锅炉泄漏 → 工人死亡 → 工程师被追责

**案例**：
```
【托马斯的失败】

托马斯，42岁，蒸汽机工程师。

他在一家纺织厂工作15年。

今天，他设计的新型蒸汽机投入使用。
这台机器能提高效率20%。

他很骄傲。

但3小时后——

轰！

蒸汽机爆炸。

锅炉压力过大，
他计算错了。

爆炸导致：
- 7名工人死亡
- 15名工人重伤
- 厂房被毁

托马斯被捕。

法院判决：
- 过失杀人罪
- 坐牢5年
- 赔偿5000币

他出狱时，47岁。

没人敢雇佣他。

他只能去修理铺，
当学徒。
月薪50币。

从中产跌落到贫民。

一次失败，毁了一生。
```

---

#### **社会地位**

**受尊敬**：
- 技术专家
- 推动工业发展
- 收入稳定

**但压力大**：
- 必须不断学习
- 事故责任重大
- 工作强度高

### **3. 教师（Teacher）**

**基本信息**：
- **年龄**：22-60岁
- **月收入**：150-400币
- **工作地点**：公立学校、私立学校、家教
- **学历要求**：师范学院毕业（4年，学费2000币）

---

#### **职业细分**

**A. 公立小学教师**
- **月收入**：150-200币
- **学生**：工人阶级子弟
- **班级规模**：50-80人
- **工作强度**：高

**B. 私立学校教师**
- **月收入**：250-400币
- **学生**：中产、富人子弟
- **班级规模**：15-30人
- **工作强度**：中等

**C. 私人家教**
- **月收入**：300-800币（看雇主）
- **学生**：贵族、大商人子女
- **教学内容**：全科+礼仪+才艺
- **社会地位**：高

---

#### **工作内容**

**典型一天（公立小学教师）**：
```
06:30 - 起床
07:00 - 早餐（硬面包+咖啡，20分）
07:30 - 到学校
08:00-12:00 - 上课（4节课）
  - 语文（识字、写字）
  - 数学（加减乘除）
  - 历史（帝国荣光）
  - 纪律管理（打手心）
12:00-12:30 - 午餐（学校食堂，30分）
12:30-16:00 - 继续上课（3节课）
16:00-17:00 - 批改作业
17:00 - 回家
17:30 - 晚餐（家里，便宜）
18:00 - 备课
20:00 - 睡觉

月收入：180币（固定工资）
```

---

#### **职业困境**

**收入低**：
- 公立教师工资仅略高于工人
- 私立教师稍好
- 家教收入高但不稳定

**压力大**：
- 学生多（50-80人）
- 管不住
- 家长投诉
- 校长责骂

**社会地位矛盾**：
- 理论上"受尊敬"
- 实际上"穷书生"
- 被富人轻视

**案例**：
```
【艾米莉老师的绝望】

艾米莉，28岁，公立小学教师。

她教一个班，68个学生。

这些学生：
- 30%文盲（父母不识字）
- 50%经常缺课（要打工）
- 80%衣衫褴褛（买不起制服）

她每天喊哑嗓子。

她每月赚180币。

扣除：
- 房租：50币
- 食物：60币
- 衣物：20币
- 其他：30币

结余：20币。

她想存钱，去私立学校。
但私立学校要求：
- 至少5年教学经验（她才3年）
- 推荐信（她没有关系）
- "体面"的外表（她穿不起礼服）

她存了3年，攒了700币。

今年，她生病了。
肺炎。
治疗费：300币。

她的积蓄，又花了一半。

她今年31岁了。

她还能等多久？

她开始考虑：
要不，找个男人嫁了算了。
即使他只是个工人。

至少，不用这么累。
```

### **4. 记者（Journalist）**

**基本信息**：
- **年龄**：22-50岁
- **月收入**：200-600币
- **工作地点**：报社、街头
- **学历要求**：识字即可，但大学毕业更好

---

#### **职业细分**

**A. 底层记者（Reporter）**
- **月收入**：200-300币
- **工作**：跑新闻、写稿
- **危险**：被打、被威胁

**B. 调查记者（Investigative Journalist）**
- **月收入**：400-600币
- **工作**：深度调查（腐败、犯罪）
- **危险**：被暗杀

**C. 御用记者（Propaganda Writer）**
- **月收入**：500-800币
- **工作**：为政府/贵族写软文
- **道德成本**：出卖良心

---

#### **工作内容**

**典型一天（底层记者）**：
```
06:00 - 起床
06:30 - 去码头（听说有大新闻）
07:00 - 采访码头工人
  "昨晚发生了什么？"
  "有5个工人被货物砸死。"
  "工头有说什么吗？"
  "他说是工人自己不小心。"
08:00 - 去医院采访伤者
09:00 - 回报社
10:00-12:00 - 写稿
  "码头惨案：5人死亡，安全谁负责？"
12:00 - 午餐（街头小吃，20分）
13:00 - 主编看稿
  "写得太激进了，改。"
  "把'谁负责'删掉。"
  "标题改成'码头事故，令人惋惜'。"
14:00-16:00 - 改稿（无奈）
16:00 - 继续跑新闻
  - 工厂罢工（被赶）
  - 贵族宴会（进不去）
  - 街头打架（写了）
18:00 - 回报社
19:00 - 下班
19:30 - 晚餐（便宜酒馆，50分）
20:00 - 喝朗姆酒（忘记今天的屈辱）
22:00 - 回家睡觉

月收入：250币
```

---

#### **职业风险**

**被威胁**：
- 揭露腐败 → 被警告"别多管闲事"
- 继续写 → 被打、被抓

**被收买**：
- 富人给钱："别写了，我给你500币。"
- 拒绝 → 失去500币
- 接受 → 失去良心

**被暗杀**：
- 调查记者死亡率：10%/年
- 官方说法：意外、自杀
- 实际：被灭口

**案例**：
```
【威廉记者的最后一篇稿】

威廉，35岁，调查记者。

他在调查一起腐败案：
某帝国高官贪污100万币。

他花了3个月，收集证据：
- 银行记录（偷拍）
- 证人口供（冒险采访）
- 账本（贿赂会计得到）

他写了一篇10000字的深度报道。

报社主编看了：
"威廉，这篇不能发。"

"为什么？"

"那个高官，是军团将军的亲戚。"

"所以？"

"所以，发了，我们报社会被查封。"

威廉沉默了。

他拿着稿子，离开报社。

他去了另一家报社。

第二家报社也拒绝了。

第三家报社也拒绝了。

没有人敢发。

一周后，
威廉在家中被发现。

死因：煤气中毒。

官方结论：意外。

但他的同事知道：
他家没有煤气。

他的稿子，消失了。

没人再提那个贪污案。

一切如常。
```

---

#### **社会地位**

**矛盾的存在**：
- 理论上：监督权力、传播真相
- 实际上：被权力控制、传播谎言

**记者的三种结局**：
1. **继续抗争** → 被暗杀
2. **被收买** → 成为御用记者（富裕但良心不安）
3. **放弃理想** → 只写娱乐新闻（平庸但安全）

### **5. 小商人（Small Business Owner）**

**基本信息**：
- **年龄**：25-60岁
- **月收入**：200-1000币（极不稳定）
- **工作地点**：自己的店铺
- **启动资金**：500-3000币

---

#### **职业类型**

**A. 杂货店老板**
- **启动资金**：500-1000币
- **月收入**：200-400币
- **利润率**：10-20%

**B. 服装店老板**
- **启动资金**：1000-2000币
- **月收入**：300-600币
- **利润率**：30-40%

**C. 餐馆老板**
- **启动资金**：1500-3000币
- **月收入**：400-1000币
- **利润率**：20-30%
- **风险**：高（食材损耗、卫生问题）

---

#### **工作内容**

**典型一天（杂货店老板）**：
```
05:00 - 起床

  - 面包卖出60个（赚60分）
  - 蔬菜卖出30斤（赚30分）
  - 盐卖出5包（赚15分）
  - 朗姆酒卖出10瓶（赚30分）
12:00-12:30 - 午餐（自己做，成本10分）
12:30-18:00 - 继续卖货
  - 面包卖出30个（赚30分）
  - 蔬菜卖出15斤（赚15分）
```
