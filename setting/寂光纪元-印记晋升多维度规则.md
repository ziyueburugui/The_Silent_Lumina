# 《寂光纪元》印记晋升多维度规则体系

## **⟦ 核心理念 ⟧**

> **晋升不只是材料+仪式**  
> **晋升是一个"条件组合"的过程**  
> **  
> **每道印记的刻印需要满足：**  
> **1. 物质条件（材料、环境）**  
> **2. 精神条件（理智、认知）**  
> **3. 时机条件（时间、事件）**  
> **4. 代价条件（牺牲、交换）**  
> **5. 共鸣条件（与已有印记的关系）**

---

## **⟦ Ⅰ. 第1道印记：被动标记 ⟧**

### **刻印条件**

这道印记是自然形成的，但形成的**类型**取决于：

#### **1. 接触的异常类型**
- 影子异常 → 影之印记
- 火焰异常 → 焰之印记
- 深海异常 → 深渊印记

#### **2. 接触时的精神状态**
```
恐惧状态 → 印记更"野性"、难控制
冷静状态 → 印记更"稳定"、易控制
崩溃状态 → 可能直接失控

示例：
同样接触影仆
- 恐惧者：获得"狂暴影之印记"（更强但更难控）
- 冷静者：获得"稳定影之印记"（较弱但易控）
```

#### **3. 接触的环境**
```
在黑暗中接触 → 印记"亲暗"
在水中接触 → 印记"亲水"
在裂隙界接触 → 印记更"不稳定"

这会影响后续晋升的方向
```

#### **4. 是否"主动"还是"被动"**
```
主动接触（如守夜人任务）→ 印记较"驯服"
被动遭遇（如意外）→ 印记较"野性"

驯服的印记：容易晋升第2道
野性的印记：难晋升，但潜力更大
```

**所以第1道印记虽然是被动的，但有"品质"之分！**

## **⟦ Ⅱ. 第2道印记：共鸣刻印 ⟧**

### **核心概念：印记之间的"共鸣"**

第2道印记不是随便选的，它必须与第1道印记"共鸣"

---

### **刻印条件组合**

#### **【主要条件】污染获取**
- 必须在任务中接触第二种异常
- 活下来并获得污染

#### **【次要条件1】印记兼容性**
不是所有印记都能共存！

**相生关系**（成功率+20%）：
```
影 + 冰 = 好（都是"寒冷系"）
火 + 雷 = 好（都是"能量系"）
水 + 风 = 好（都是"流动系"）
```

**相克关系**（成功率-30%）：
```
火 + 冰 = 差（互相冲突）
光 + 暗 = 差（天生对立）
```

**中立关系**（标准成功率）：
```
影 + 地 = 一般
火 + 水 = 一般（可以共存但不协同）
```

#### **【次要条件2】刻印时机**
不是随时都能刻！

**最佳时机**（成功率+10%）：
```
满月之夜（对某些印记）
新月之夜（对另一些印记）
寂光夜周期的特殊节点
裂隙界"潮汐"高峰期
```

**示例**：
```
影之印记 → 新月刻印最佳
光明印记 → 满月刻印最佳
深渊印记 → 深海"潮汐"高峰最佳
```

**错误时机**（成功率-20%）：
```
在错误的时间刻印
如：在满月刻暗影印记
```

#### **【次要条件3】刻印环境**
必须在"适合"的环境！

**环境匹配**（成功率+15%）：
```
影之印记 → 在完全黑暗的房间
火之印记 → 在高温熔炉旁
深渊印记 → 在海边或水下
时间印记 → 在古老的钟塔中
```

**环境错误**（成功率-25%）：
```
在强光下刻影之印记 → 极度困难
在水中刻火之印记 → 几乎不可能
```

#### **【次要条件4】理智阈值**
```
理智太低 → 失控风险高
理智太高 → 印记"排斥"

最佳理智值：85-90%
低于80% → 成功率-15%
低于75% → 成功率-30%
```

#### **【次要条件5】"仪式代价"**
第2道开始需要"象征性牺牲"

**示例**：
```
刻影之印记 → 必须在刻印时"放弃光明"
- 具体：在刻印后的一周内不能见阳光
- 代表：接纳黑暗

刻焰之印记 → 必须"燃烧珍贵之物"
- 具体：烧掉一件对你最重要的东西
- 代表：破坏重生

刻深渊印记 → 必须"沉入深海"
- 具体：在深海（100米以下）待1小时
- 代表：承受压力

刻时间印记 → 必须"失去时间"
- 具体：进入"时间停滞区"待24小时
- 对外界：你"消失"了一天
- 代表：脱离时间流
```

这些代价是**象征性的**，但必须完成！

## **⟦ Ⅲ. 第3道印记：材料融合 ⟧**

### **核心概念：不只是"有材料"就行**

第3道开始，材料的"品质"、"新鲜度"、"来源"都很重要

---

### **刻印条件组合**

#### **【主要条件】异常材料**
- C级仆从核心
- 或Euclid级SCP部件
- 或裂隙界第三层物质

**但材料有"品质"！**

**上品材料**（成功率+15%）：
```
- 刚死亡的仆从（24小时内）
- 核心完整无损
- 来自"纯血"仆从（未混血）

示例：
你刚杀死一只C级影仆
立即提取影核
核心完整、新鲜
→ 上品材料
```

**中品材料**（标准）：
```
- 死亡3-7天
- 核心有轻微损伤
- 来自普通仆从
```

**下品材料**（成功率-20%）：
```
- 死亡超过7天
- 核心破损
- 来自病弱/老迈仆从
- 或从黑市购买的（赛忒拉常见）
```

#### **【次要条件1】材料处理**
材料不能直接用！必须"净化"

**净化流程**（7天）：
```
第1-3天：浸泡在圣水/药剂中
→ 去除"杂质污染"

第4-5天：在月光/星光下曝晒
→ "激活"材料

第6天：用自己的血液浸泡
→ "调谐"到你的频率

第7天：材料"准备就绪"
```

**如果跳过净化**：
- 成功率-40%
- 污染加剧
- 可能异化

#### **【次要条件2】印记"平衡"**
第3道开始要注意"印记平衡"

**平衡原则**：
```
不能让一个"元素"太多

示例：
第1道：影（暗系）
第2道：冰（寒系）
第3道：不能再选暗/寒系！
→ 应该选择相反或中立的
→ 如：火（热系）或土（中立）

如果强行选暗/寒系：
→ 身体会"极化"
→ 成功率-30%
→ 即使成功，副作用加倍
```

**推荐配置**：
```
暗-寒-热（平衡三角）
影-冰-火

或

能量-物质-精神（三大类平衡）
雷-地-心灵
```

#### **【次要条件3】"场域"要求**
第3道必须在"特殊场域"中刻印

**什么是场域？**
```
异常能量密集的地点：
- 裂隙口附近（1公里内）
- 古老的异常收容设施
- 寂光夜遗迹
- 旧日降临点
```

**为什么需要场域？**
```
第3道印记需要"外部能量"辅助
普通环境的能量不够
必须借助场域的"异常浓度"
```

**场域强度**：
```
强场域（成功率+20%）：
- 活跃裂隙口旁
- 旧日降临点

中场域（标准）：
- 稳定裂隙口
- 守夜人收容设施

弱场域（成功率-15%）：
- 远离裂隙的普通地点
```

#### **【次要条件4】"共鸣者"在场**
第3道需要"共鸣者"辅助

**共鸣者定义**：
```
至少拥有3道印记的人
并且其中一道与你要刻的印记"同类"

示例：
你要刻"影之印记"
共鸣者必须也有影之印记
```

**共鸣者的作用**：
```
在刻印时：
- 共鸣者站在旁边
- 激活自己的同类印记
- 产生"共鸣场"
- 引导你的刻印

效果：成功率+15%
```

**如果没有共鸣者**：
- 可以刻，但成功率-20%

#### **【次要条件5】"契约仪式"**
第3道开始需要"契约"

**契约对象**：
```
选项1：与"印记本身"立契约
- 承诺：用它的力量守护XX
- 如果违背：印记"反噬"

选项2：与"异常实体"立契约
- 如：与影之领主立约
- 承诺：为它做XX事
- 如果违背：被追杀

选项3：与"组织"立契约
- 如：与守夜人立约
- 承诺：服役XX年
- 如果违背：被通缉
```

**契约是强制的**，选一个

**为什么需要契约？**
```
第3道开始，印记的力量"过强"
需要"约束"来平衡
契约就是这种约束

没有契约 = 力量失控 = 必然异化
```
