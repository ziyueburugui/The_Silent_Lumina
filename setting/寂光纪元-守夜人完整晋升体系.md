# 《寂光纪元》守夜人完整晋升体系

## **⟦ 守夜人晋升总览 ⟧**

### **核心原则**

> **守夜人是最保守的组织**  
> **每一次晋升都需要严格审核**  
> **因为他们最在乎"人性"**  
> **而不仅仅是"力量"**

---

## **⟦ 第1道印记：被动获得 ⟧**

### **获取条件**

**唯一条件**：
- 接触异常并活下来
- 7天后自动形成
- **无需申请，无需审核**

**守夜人的应对**：
1. 监控系统自动标记疑似污染者
2. 派遣"接触者"前往观察
3. 第7天前完成初步接触
4. 第7天确认印记形成
5. 邀请加入守夜人

**邀请话术**：
```
"你已经被标记了。"
"你有两个选择："
"加入我们，学习如何活下去。"
"或者独自面对，90%会在半年内死亡。"
"考虑一下吧。"
```

**加入流程**：
```
第1周：观察期
→ 住在守夜人宿舍
→ 接受心理评估
→ 学习基础知识

第2周：培训期
→ 学习如何控制印记
→ 学习如何识别异常
→ 学习如何战斗

第3周：考核期
→ 模拟任务
→ 实战考核
→ 通过 = 正式Ⅰ级守夜人
```

**拒绝加入的后果**：
```
"你可以拒绝。"
"但我们会监控你。"
"如果你失控，我们会处决你。"
"如果你引发事件，我们会处决你。"
"这不是威胁，这是保护其他人。"
```

## **⟦ 第2道印记：仪式刻印 ⟧**

### **晋升条件**

**基础要求**：
1. **资历要求**：成为Ⅰ级至少6个月
2. **任务要求**：完成至少20次Safe级任务，或5次Euclid级任务
3. **理智要求**：理智镜完整度≥85%
4. **污染获取**：在任务中接触新的异常并活下来
5. **导师推荐**：直属导师书面推荐
6. **心理评估**：通过专业心理评估

**申请流程**：

**第1步：提交申请**
```
向导师提交书面申请
说明：
- 为什么要晋升
- 接触了什么异常
- 想要获得什么印记
- 晋升后的计划
```

**第2步：导师评估**
```
导师会评估：
- 你的心理状态
- 你的战斗能力
- 你的理智稳定性
- 你是否"真的需要"第2道印记

如果导师认为不合适：
→ 拒绝推荐
→ 申请终止

如果导师同意：
→ 书面推荐
→ 进入下一步
```

**第3步：理智检查**
```
前往镜室
接受完整理智镜检查
医生评估：
- 完整度是否≥85%
- 核心碎片是否稳定
- 是否有异化倾向

通过 = 继续
不通过 = 申请终止
```

**第4步：心理面谈**
```
与守夜人心理医生面谈
问题包括：
"你为什么要第2道印记？"
"你知道代价吗？"
"你准备好失去更多理智了吗？"
"你的家人知道吗？"
"如果你失控，我们会杀了你。你理解吗？"

医生会评估：
- 动机是否纯正
- 心态是否稳定
- 是否理解风险

通过 = 获得批准
不通过 = 申请终止
```

**第5步：刻印仪式**
```
批准后7天内必须刻印
（污染会在7天后消退）

仪式流程：
第1-3天：准备
- 禁食
- 冥想
- 学习相关知识

第4天：刻印
- 进入刻印室
- 导师+2位Ⅱ级守夜人作为见证者
- 使用SCP-████"基础刻印仪"
- 持续10-30分钟
- 成功率：85%

第5-7天：恢复
- 休息
- 适应新印记
- 理智镜检查
```

**失败的后果**：
- 15%失控率
- 失控 = 立即被见证者处决
- **这不是玩笑**

**成功的奖励**：
- 正式晋升Ⅱ级
- 薪资提升50%
- 可以执行更高级任务
- 可以申请小队长

## **⟦ 第3道印记：材料刻印 ⟧**

### **晋升条件**

**基础要求**：
1. **资历要求**：成为Ⅱ级至少1年
2. **任务要求**：完成至少10次Euclid级任务，或参与过1次Keter级任务
3. **理智要求**：理智镜完整度≥70%
4. **材料获取**：自己猎杀异常获取"异常核心"
5. **小队长推荐**：小队长+导师双重推荐
6. **十三席之一批准**：必须有一位十三席成员批准

**材料要求**：

**必须自己获取**：
- 守夜人不提供材料
- 不能购买材料
- 不能接受赠与
- **必须亲自猎杀，亲自提取**

**为什么？**
```
"第3道印记是重要的分界线。"
"从这里开始，你要'猎杀'。"
"要用异常的'核心'强化自己。"
"这是以命换命。"
"如果你不敢杀，你不配晋升。"
```

**可接受的材料类型**：

| 材料类型 | 来源 | 难度 | 大约时间 |
|---------|------|------|---------|
| C级仆从核心 | 猎杀C级仆从 | 中等 | 1-3个月 |
| Euclid级SCP部件 | 从收容设施提取 | 较高 | 需要特殊任务 |
| 裂隙界第三层物质 | 深入裂隙界 | 高 | 3-6个月 |
| 特殊异常物品碎片 | 损坏的异常物品 | 中高 | 2-4个月 |

**申请流程**：

**第1步：提交猎杀计划**
```
书面报告，包括：
- 目标异常是什么
- 在哪里
- 如何猎杀
- 需要什么支援
- 预估时间
- 风险评估
```

**第2步：小队长+导师双重评估**
```
评估：
- 计划是否可行
- 风险是否可控
- 你是否有能力

批准 = 继续
否决 = 修改计划或放弃
```

**第3步：执行猎杀**
```
可以组队（但核心必须你亲自提取）
可以申请支援（但击杀必须你参与）

猎杀过程会被监控
守夜人会派观察员跟随
确保材料来源合法

成功 = 获得材料
失败 = 可能死亡
```

**第4步：提交材料验证**
```
将材料提交给守夜人实验室
验证：
- 是否真的是异常核心
- 是否符合要求
- 是否适合你的印记类型

通过 = 继续
不通过 = 重新猎杀
```

**第5步：十三席批准**
```
材料验证后
提交给十三席之一审核

十三席会考虑：
- 你的资历
- 你的理智状态
- 组织是否需要更多Ⅲ级
- 你是否值得投资

批准率：约60%
（很多人卡在这一步）
```

**第6步：刻印仪式**
```
批准后7天内刻印

仪式流程：
第1-7天：准备
- 禁食7天
- 冥想8小时/天
- 每天接触材料1小时（适应）
- 心理评估

第8天：刻印
- 进入高级刻印室
- 导师+2位Ⅲ级守夜人+医疗组
- 将材料放在胸口
- 启动SCP-████"高级刻印仪"
- 材料"融化"并流入身体
- 持续30-60分钟
- 极度痛苦
- 成功率：65%

第9-14天：恢复
- 住院观察
- 理智镜每天检查
- 适应新能力
```

**失败的后果**：
- 35%失败率
- 其中20%死亡，15%失控被处决
- **死亡率很高**

**成功的奖励**：
- 正式晋升Ⅲ级
- 薪资提升100%
- 成为精英战力
- 可以申请带队
- 获得更多资源
