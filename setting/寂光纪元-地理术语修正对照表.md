# 《寂光纪元》地理术语修正对照表

## **⟦ 核心问题 ⟧**

### **严重错误**

在多个文档中错误使用了现实世界（地球）的地理名称：

```
❌ 错误示例：
- "地球"
- "太平洋"、"大西洋"、"北冰洋"
- "纽约"、"伦敦"、"东京"
- "中亚"、"中东"、"北非"
- "亚马逊"、"南极"、"埃及"
- "北冰洋海底5000米"
```

**这是架空世界！不是地球！**

## **⟦ 修改原则 ⟧**

### **1. 世界层面**
```
"地球" → "这个世界" / "物质世界"
"全球" → "世界范围内" / "各大陆"
```

### **2. 海域层面**
```
"太平洋" → "拉莱耶大沟"
"大西洋" → "寂光海"
"北冰洋" → "北境无风带"
"印度洋" → "暗潮海"
```

### **3. 陆地层面**
```
现实大洲名 → 三大陆名称
现实地区名 → 具体描述位置
```

### **4. 城市层面**
```
现实城市名 → 
- 使用已设定的城市名
- 或笼统描述"大城市"、"沿海都市"
```

### **5. 自然地理**
```
现实地理名 → 描述性名称
如："亚马逊雨林" → "姆尔卡斯古老森林"
```

## **⟦ 总结 ⟧**

### **核心原则**

```
1. 这是架空世界，不是地球
2. 使用已设定的地理名称
3. 如无设定，用描述性名称
4. 绝不使用现实世界地名
```

### **关键替换**

```
地球 → 这个世界/物质世界
北冰洋 → 北境无风带
太平洋 → 拉莱耶大沟
大西洋 → 寂光海
现实城市 → 已设定城市或笼统描述
现实地区 → 三大陆 + 具体方位
```
