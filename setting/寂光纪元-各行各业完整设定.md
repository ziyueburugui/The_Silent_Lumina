```

### **1. 煤矿童工**

**基本信息**：
- **年龄**：6-14岁
- **工作时间**：每天12-16小时
- **日薪**：25-35分
- **月收入**：6.5-9币（按26天计）
- **工作地点**：赛忒拉帝国的煤矿、维拉诺斯的裂隙矿场

**工作内容**：
1. **背煤**：在狭窄的矿道里背煤袋（20-30kg）
2. **捡煤渣**：在煤堆里挑出杂质
3. **推煤车**：在轨道上推小煤车
4. **通风口守卫**：站在通风口，确保不堵塞（最危险）

**工作环境**：
- 深度：地下100-500米
- 温度：35-45°C
- 空气：充满煤尘
- 照明：微弱的煤油灯
- 危险：坍塌、瓦斯爆炸、窒息

**典型一天**：
```
04:00 - 起床（贫民窟窝棚）
04:30 - 啃一块硬面包（3分）
05:00 - 下矿井
05:00-12:00 - 背煤（7小时不停）
12:00-12:15 - 午餐（一块面包+水）
12:15-19:00 - 继续背煤
19:00 - 出矿井
19:30 - 领工钱（30分）
20:00 - 买晚餐（硬面包3分+烂菜1分）
20:30 - 回窝棚
21:00 - 睡觉

周而复始。
```

**生存状态**：
- **平均寿命**：12岁（大多数活不过10岁）
- **死亡原因**：
  - 坍塌：40%
  - 肺病：30%
  - 事故（被煤车碾压等）：20%
  - 饥饿/疾病：10%
- **死亡率**：20%/年

**真实案例**：
```
【小汤姆的故事】

小汤姆，7岁，在铁炉城3号煤矿工作。

他每天背30kg的煤袋，
走800米的矿道，
往返20次。

今天是他第137天工作。

下午3点，
矿道坍塌。

3个孩子被埋。
小汤姆是其中之一。

挖出来时，已经死了。

他手里还握着一个煤袋。

工头赔了5币给他妈妈。

第二天，
工头又招了3个新童工。

生活继续。
```

**职业特点**：
- ✅ 门槛极低（只要能走路）
- ✅ 到处招人（死亡率高，缺人）
- ❌ 极度危险
- ❌ 无未来
- ❌ 工头经常克扣工资/发假币

### **2. 纺织厂童工**

**基本信息**：
- **年龄**：5-14岁
- **工作时间**：每天14-16小时
- **日薪**：20-30分
- **月收入**：5-8币
- **工作地点**：赛忒拉帝国、塔纳托斯联邦的纺织厂

**工作内容**：
1. **捡线头**：在机器下捡掉落的线头
2. **清理机器**：钻进机器底下清理棉絮
3. **换纱线轴**：给织布机换新纱线
4. **质检**：检查布料有无瑕疵

**工作环境**：
- 温度：30-40°C（蒸汽机散热）
- 噪音：100-120分贝（织布机轰鸣）
- 空气：充满棉絮
- 照明：昏暗（省灯油）
- 危险：被机器夹断手指

**典型一天**：
```
05:00 - 起床
05:30 - 吃早餐（硬面包3分）
06:00 - 进厂
06:00-12:00 - 捡线头
12:00-12:10 - 午餐（面包+水）
12:10-20:00 - 继续工作
20:00 - 下班
20:30 - 买晚餐（硬面包3分）
21:00 - 睡觉
```

**生存状态**：
- **平均寿命**：15岁
- **死亡原因**：
  - 机器夹伤：30%
  - 肺病（吸入棉絮）：50%
  - 过劳死：15%
  - 其他：5%
- **死亡率**：15%/年
- **致残率**：30%/年（断指、断手）

**真实案例**：
```
【小玛丽的手】

小玛丽，9岁，纺织厂童工。

今天，她在清理织布机时，
袖子被卷进去。

她尖叫。

工头按下紧急停机。

但已经晚了。

她的右手，
从手腕处被扯断。

工头给了她2币"赔偿"。
然后把她抬出去。

"你不能再在这工作了。"

小玛丽失去了：
- 右手
- 工作
- 未来

她现在只能：
- 乞讨
- 或者死

她选择了第二个。

一周后，
她在河里被找到。
```

**职业特点**：
- ✅ 比煤矿安全（相对）
- ✅ 有"工伤赔偿"（虽然少得可怜）
- ❌ 噪音导致失聪
- ❌ 棉絮导致肺病
- ❌ 致残率高

### **3. 街头报童**

**基本信息**：
- **年龄**：6-16岁
- **工作时间**：从早到晚（不固定）
- **日收入**：10-40分（看运气）
- **月收入**：3-10币（极不稳定）
- **工作地点**：各大城市街头

**工作内容**：
1. **卖报纸**：从报社批发报纸（4分/份），卖5分/份，赚1分差价
2. **卖火柴**：2分/盒，卖3分/盒
3. **擦鞋**：5-10分/次
4. **送信**：10-20分/次

**工作环境**：
- 地点：街头、广场、车站
- 天气：风吹日晒雨淋
- 竞争：其他报童抢地盘
- 危险：被流氓抢劫、被马车撞

**典型一天**：
```
05:00 - 起床
05:30 - 去报社批发20份报纸（80分）
06:00 - 开始叫卖："卖报纸！今日大新闻！"
06:00-12:00 - 卖掉15份（赚15分）
12:00 - 用10分买半块硬面包
12:30 - 继续卖
12:30-18:00 - 卖掉5份（赚5分）
18:00 - 回家，今天赚了20分

好日子：卖完20份，赚20分
坏日子：只卖10份，亏损30分（报纸不退货）
```

**生存状态**：
- **平均寿命**：20岁
- **死亡原因**：
  - 饿死：35%
  - 冻死（冬天）：25%
  - 被打死（抢地盘/被抢劫）：20%
  - 疾病：15%
  - 被车撞：5%
- **死亡率**：25%/年

**真实案例**：
```
【小杰克的冬天】

小杰克，10岁，报童。

今年冬天特别冷。
气温-15°C。

小杰克穿着破布衣服，
站在街角卖报纸。

没人买。
大家都躲在家里。

他站了8小时。
一份都没卖出去。

晚上，他找了个墙角，
蜷缩着睡觉。

第二天早上，
清洁工发现了他。

已经冻死了。

手里还握着那20份报纸。

没人认领他的尸体。

他被扔进乱葬岗。

和其他300个尸体一起。
```

**职业特点**：
- ✅ 自由（没有工头）
- ✅ 收入全归自己
- ❌ 极度不稳定
- ❌ 冬天几乎饿死
- ❌ 竞争激烈（地盘战）

### **4. 齿轮工厂打磨工**

**基本信息**：
- **年龄**：8-16岁
- **工作时间**：每天12-14小时
- **日薪**：25-35分
- **月收入**：6.5-9币
- **工作地点**：赛忒拉的机械工厂

**工作内容**：
1. **打磨齿轮**：用砂纸打磨齿轮，去除毛刺
2. **上油**：给齿轮涂上防锈油
3. **质检**：检查齿轮是否合格
4. **包装**：把齿轮装箱

**工作环境**：
- 温度：25-35°C
- 噪音：80-100分贝
- 空气：金属粉尘
- 照明：较好（需要精细工作）
- 危险：金属粉尘导致肺病、手指磨损

**典型一天**：
```
06:00 - 起床
06:30 - 早餐（硬面包3分）
07:00 - 进厂
07:00-12:00 - 打磨齿轮（500个）
12:00-12:15 - 午餐（面包+汤，10分）
12:15-19:00 - 继续打磨（500个）
19:00 - 下班
19:30 - 晚餐（硬面包3分+菜1分）
20:00 - 睡觉

每天打磨1000个齿轮。
手指磨出血泡、结痂、再磨破。
```

**生存状态**：
- **平均寿命**：18岁
- **死亡原因**：
  - 肺病（金属粉尘）：60%
  - 过劳死：20%
  - 事故（被机器压死）：15%
  - 其他：5%
- **死亡率**：12%/年

**真实案例**：
```
【阿伦的手指】

阿伦，12岁，打磨工。

他在这工作3年了。

他的手指，
没有指纹了。
被砂纸磨平了。

他的指甲，
大部分掉了。

他的手掌，
全是厚厚的茧。

但他还在打磨。
每天1000个齿轮。

工头说：
"你做得很好。
再过5年，你就能升为熟练工。
月薪40币。"

但阿伦知道，
他活不了5年。

他已经开始咳血了。
金属粉尘在他肺里。

他可能还有2年。

但他继续打磨。
因为他弟弟还要吃饭。
```

**职业特点**：
- ✅ 比煤矿和纺织厂稍好
- ✅ 有"晋升"可能（成为熟练工）
- ❌ 手指永久损伤
- ❌ 金属粉尘导致肺病
- ❌ 需要眼力和手感（老了就没用）

### **5. 学徒（铁匠/木匠/裁缝）**

**基本信息**：
- **年龄**：10-18岁
- **学徒期**：3-5年
- **月薪**：8-15币（学徒期）
- **月薪**：40-80币（出师后）
- **拜师费**：10-50币（需要家里攒几年）

**工作内容**：

**铁匠学徒**：
1. **拉风箱**：保持炉火旺盛
2. **递工具**：给师傅递锤子、钳子
3. **打下手**：敲打初步成型
4. **清理**：打扫铺子

**木匠学徒**：
1. **刨木板**：初步处理木材
2. **递工具**：给师傅递锯子、凿子
3. **打磨**：打磨木制品
4. **清理**：清扫木屑

**裁缝学徒**：
1. **缝边**：缝制衣服边角
2. **熨烫**：熨平布料
3. **剪线头**：清理多余线头
4. **跑腿**：送货、买布料

**工作环境**：
- 地点：师傅的铺子/家
- 时间：每天10-14小时
- 待遇：包吃包住（有些师傅）或自己解决
- 学习：边干边学

**典型一天（铁匠学徒）**：
```
05:00 - 起床（睡在铺子后面）
05:30 - 生火、准备工具
06:00 - 师傅起床
06:30-12:00 - 拉风箱、递工具、学习
12:00-12:30 - 午餐（师傅提供，或自己买）
12:30-18:00 - 继续工作、学习
18:00-19:00 - 清理铺子
19:00 - 晚餐
19:30 - 自己练习（如果还有力气）
20:30 - 睡觉
```

**生存状态**：
- **平均寿命**：35岁（出师后）
- **学徒期死亡率**：15%/3年
- **死亡原因**（学徒期）：
  - 工伤（烧伤、砸伤）：60%
  - 过劳死：30%
  - 疾病：10%

**真实案例**：
```
【艾伦的梦想】

艾伦，12岁，铁匠学徒。

他爹攒了3年，
凑了20币拜师费。

师傅是个老铁匠，
技术好，但脾气暴。

艾伦每天拉风箱12小时。
手上全是水泡。

水泡破了、结痂、再起泡。

师傅说：
"熬过5年，你就能当铁匠。
月薪60币。"

艾伦咬牙坚持。

3年后，
他成为了二年级学徒。
开始学打铁。

5年后，
他出师了。
月薪60币。

10年后，
他自己开了铺子。
月薪100币。

15年后，
他收了3个学徒。

他成功了。

但他看着自己的学徒，
想起了当年的自己。

水泡、结痂、血。

"这就是命。"
```

**职业特点**：
- ✅ 有"未来"（出师后收入不错）
- ✅ 学到技能（铁饭碗）
- ✅ 社会地位比童工高
- ❌ 拜师费昂贵（普通家庭要攒3年）
- ❌ 学徒期很苦
- ❌ 师傅可能虐待
- ❌ 15%死亡率

### **6. 妓女/酒馆侍女（贫民窟）**

**基本信息**：
- **年龄**：12-30岁
- **工作时间**：晚上6点-凌晨2点
- **名义收入**：30-80币/月
- **实际到手**：15-30币/月（老鸨抽70%）
- **工作地点**：贫民窟妓院、肮脏酒馆

**工作内容**：

**妓女**：
1. **接客**：每晚5-10个客人
2. **收费**：10-50分/次（看长相和年龄）
3. **上交**：70%给老鸨

**酒馆侍女**：
1. **端酒**：给客人端朗姆酒、啤酒
2. **陪酒**：陪客人聊天、喝酒
3. **"额外服务"**：有些酒馆要求

**工作环境**：
- 地点：贫民窟妓院、破旧酒馆
- 卫生：极差（没有清洁措施）
- 客人：码头工人、矿工、流氓
- 危险：性病、暴力、酗酒

**典型一天（妓女）**：
```
12:00 - 起床（晚上工作，白天睡觉）
13:00 - 午餐（5分的硬面包+菜）
14:00 - "化妆"（用便宜胭脂）
15:00 - 去妓院
18:00 - 开始接客
18:00-02:00 - 接客7人（每人30分，共2.1币）
02:00 - 上交给老鸨（2.1币 × 70% = 1.47币）
02:00 - 到手60分
03:00 - 回窝棚睡觉
```

**生存状态**：
- **平均寿命**：25岁
- **死亡原因**：
  - 性病：40%
  - 暴力（被客人打死）：30%
  - 自杀：20%
  - 其他：10%
- **死亡率**：30%/年

**真实案例**：
```
【莉莉的三年】

莉莉，15岁，妓女。

她原本是纺织厂童工。
但她12岁时，手被机器夹断。

她失去工作。
她妈妈病了，需要钱。

她没有选择。

第一年：
她每晚接客5人。
收入1.5币/天，到手45分。
月收入13.5币。

她妈妈的药费：10币/月。
她自己的开支：5币/月。

刚好够。

第二年：
她得了性病。
治疗费：5币。
她找黑市医生。

但治不好。
只是缓解。

她开始咳嗽、发烧。

第三年：
她18岁了。
"老"了。

客人越来越少。
收入下降到8币/月。

不够。

她开始借高利贷。

一个月后，
她在河里被发现。

死因：溺水。

实际上是自杀。

她留下一封信：

"对不起，妈妈。
我不想活了。
这个世界太脏了。"
```

**职业特点**：
- ✅ 收入相对高（对文盲女性而言）
- ✅ 门槛低（只要年轻）
- ❌ 性病几乎必得
- ❌ 暴力风险极高
- ❌ 社会地位最低
- ❌ 平均寿命极短

### **7. 搬运工**

**基本信息**：
- **年龄**：16-50岁
- **工作时间**：每天10-14小时（看货物量）
- **日薪**：60-80分
- **月收入**：15-20币
- **工作地点**：码头、火车站、仓库

**工作内容**：
1. **搬货**：搬运货物（20-80kg/件）
2. **装卸**：装卸火车、船只
3. **分类**：把货物分类堆放
4. **清点**：清点货物数量

**工作环境**：
- 地点：露天码头、仓库
- 天气：风吹日晒雨淋
- 重量：每天搬运2-5吨货物
- 危险：砸伤、摔伤

**典型一天**：
```

一天搬6吨，背都直不起来。

  - 过劳死（心脏病）：50%

老杰克，45岁，搬运工。

### **8. 清洁工**

**基本信息**：
- **年龄**：20-60岁（多为女性或老人）
- **工作时间**：每天8-12小时
- **日薪**：50-70分
- **月收入**：13-18币
- **工作地点**：街道、工厂、富人家

**工作内容**：
1. **扫街**：清扫街道垃圾
2. **倒垃圾**：收集并倾倒垃圾
3. **清理下水道**：疏通堵塞的下水道
4. **清洁公共厕所**：最脏最累的工作

**工作环境**：
- 地点：街头、垃圾场、下水道
- 气味：恶臭
- 卫生：极差
- 危险：疾病（鼠疫、霍乱）

**典型一天**：
```
04:00 - 起床
04:30 - 早餐（硬面包3分）
05:00 - 开始扫街
05:00-12:00 - 清扫5条街道
12:00-12:15 - 午餐（面包+水）
12:15-14:00 - 倒垃圾
14:00-17:00 - 清理下水道
17:00 - 下班
17:30 - 领工钱（60分）
18:00 - 晚餐（面包+菜，10分）
19:00 - 睡觉

满身臭味，但至少活着。
```

**生存状态**：
- **平均寿命**：45岁
- **死亡原因**：
  - 疾病（接触病菌）：70%
  - 事故（掉进下水道）：15%
  - 过劳死：10%
  - 其他：5%
- **死亡率**：8%/年

**真实案例**：
```
【老玛丽的手】

老玛丽，58岁，清洁工。

她干了40年清洁。

她每天清理：
- 5条街道的垃圾
- 30个垃圾桶
- 3个下水道

她的手：
- 粗糙得像树皮
- 指甲缝永远是黑的
- 洗不掉

她的身上：
- 永远有臭味
- 洗不掉
- 即使用肥皂

她的邻居躲着她。
她的孙子不敢靠近她。

但她还在干。
因为她需要18币养活自己。

今年冬天，
她感染了肺炎。

庸医收费5币。
她没钱。

她死在窝棚里。

没人为她哭泣。
因为她太臭了。

她被草草埋在郊外。

她的孙女，10岁，
继承了她的工作。

循环继续。

**职业特点**：
- ✅ 门槛极低（老人、女性都能做）
- ✅ 相对稳定（总要有人扫街）
- ❌ 社会地位极低
- ❌ 疾病风险高
- ❌ 臭味洗不掉

### **9. 门卫/守夜人（非印记者）**

**基本信息**：
- **年龄**：30-65岁（多为退伍军人或老工人）
- **工作时间**：12小时轮班（白班/夜班）
- **月薪**：20-28币
- **工作地点**：工厂、仓库、富人家、商铺

**工作内容**：
1. **看门**：检查进出人员
2. **巡逻**：定时巡逻检查
3. **记录**：登记访客
4. **报警**：发现异常时报警

**工作环境**：
- 地点：门口或门房
- 设施：小凳子、煤油灯
- 气候：露天或半露天
- 危险：遇到小偷/强盗

**典型一天（夜班）**：
```
18:00 - 到岗
18:00-06:00 - 值班（12小时）
期间：
- 每小时巡逻一次
- 记录进出人员
- 喝朗姆酒取暖（冬天）
- 打瞌睡（偷偷的）
06:00 - 下班
06:30 - 回家
07:00 - 吃早餐
08:00 - 睡觉
14:00 - 起床
15:00 - 午餐+晚餐合并
17:30 - 出门上班
```

**生存状态**：
- **平均寿命**：55岁
- **死亡原因**：
  - 老年病：60%
  - 被小偷杀死：20%
  - 冻死（冬天值夜班）：15%
  - 其他：5%
- **死亡率**：5%/年

**真实案例**：
```
【老兵汤姆的夜】

汤姆，52岁，门卫。

他曾是赛忒拉帝国的士兵。
参加过边境战争。

喝着劣质朗姆酒取暖。

凌晨2点，

### **10. 马车夫**

**基本信息**：
- **年龄**：20-55岁
- **工作时间**：每天10-14小时
- **日收入**：80分-1.2币（看生意）
- **月收入**：20-30币
- **工作地点**：城市街道

**工作模式**：
1. **私人雇佣**：月薪25-30币，给富人家赶车
2. **公共马车**：每趟5-10分，自己赚差价
3. **租马车**：租金5币/月，收入归自己

**工作内容**：
1. **赶车**：驾驭马车运送乘客/货物
2. **照顾马**：喂马、刷马、清理马厩
3. **维修**：修理简单的马车故障
4. **收费**：向乘客收费

**工作环境**：
- 地点：街头
- 天气：风吹日晒雨淋
- 危险：马惊、车祸
- 竞争：其他马车夫抢客

**典型一天（公共马车）**：
```
05:00 - 起床
05:30 - 去马厩（租的）
06:00 - 喂马、刷马、套车
07:00 - 开始拉客
07:00-20:00 - 拉20趟（每趟平均5分，共1币）
期间：
- 午餐：在街头买（15分）
- 喂马：买草料（10分）
20:00 - 收车
20:30 - 卸马、刷马、清理
21:00 - 回家
21:30 - 晚餐（20分）
22:00 - 睡觉

一天收入1币，扣除：
- 马厩租金：20分/天
- 草料：10分/天
- 餐费：35分/天
- 净收入：35分/天
- 月收入：约10币

如果生意好：月收入20-25币
```

**生存状态**：
- **平均寿命**：50岁
- **死亡原因**：
  - 车祸：40%
  - 被马踢死/拖死：25%
  - 过劳死：20%
  - 其他：15%
- **死亡率**：7%/年

**真实案例**：
```
【老彼得的最后一程】

老彼得，48岁，马车夫。

他干了25年。

他的马叫"老棕"，
陪了他10年。

今天，他拉了第18趟。

突然，
一辆蒸汽车失控，
冲向他的马车。

"老棕！"

老彼得拉缰绳。

老棕想躲，
但来不及。

蒸汽车撞上马车。

老棕当场死亡。
老彼得被甩出去，
头撞在石头上。

他躺在地上，
看着老棕的尸体。

他伸出手，
想摸摸它。

但够不到。

他死了。

蒸汽车司机赔了20币。

老彼得的儿子用这钱，
买了一匹新马。

继续当马车夫。

循环继续。
```

**职业特点**：
- ✅ 收入比搬运工稍高
- ✅ 相对自由（公共马车）
- ✅ 能见到城市各个角落
- ❌ 需要启动资金（租马车）
- ❌ 蒸汽车出现后竞争加剧
- ❌ 车祸风险高
