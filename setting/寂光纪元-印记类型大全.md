# 《寂光纪元》印记类型大全

## **⟦ 印记分类系统 ⟧**

### **按来源分类**

```
基础印记（常见）
→ 来自常见异常
→ 数量：约20种
→ 获取：相对容易

稀有印记（罕见）
→ 来自特殊异常
→ 数量：约15种
→ 获取：困难

传说印记（极稀有）
→ 来自强大异常
→ 数量：约10种
→ 获取：极度困难

禁忌印记（不该存在）
→ 来自禁忌存在
→ 数量：约5种
→ 获取：几乎不可能

概念印记（未知）
→ 来自不可名状之物
→ 数量：？？？
→ 获取：？？？
```

## **⟦ Ⅰ. 基础印记（20种）⟧**

### **1. 影之印记**

**来源**：
- 影仆、暗影生物
- SCP-████（影子异常）
- 裂隙界第二层的影子区域

**外观**：
- 黑色流动纹路
- 位置：左臂内侧
- 特征：在黑暗中会发出黑光
- 看起来：活着的影子在皮肤下游走

**被动能力：暗影视觉**
- 效果：在完全黑暗中清晰视物
- 范围：30米
- 代价：在强光下会头痛

**主动能力：影之融合**
- 效果：融入阴影30秒，完全隐形
- 消耗：每次破碎1片感知碎片
- 冷却：1小时
- 限制：必须有阴影，不能在强光下使用

**永久代价**：
- 影子开始"独立"
- 有时与你的动作不同步
- 在阳光下会感到不适
- 逐渐变成"夜行性"

**第2道材料需求**：
- C级影仆的影核

**第3道材料需求**：
- B级影领主的暗核

**守夜人评价**：
```
"影之印记是最常见的印记之一。"
"大约25%的守夜人有这个印记。"
"优点：隐蔽、侦查、暗杀。"
"缺点：惧光、影子失控、夜行性。"
"适合：侦查员、刺客型守夜人。"
```

### **2. 焰之印记**

**来源**：
- 焰仆、火焰生物
- SCP-████（火焰异常）
- 裂隙界火焰区域

**外观**：
- 红色燃烧图案
- 位置：右臂外侧
- 特征：体温升高时会发热发光
- 看起来：火焰的纹身

**被动能力：高温抗性**
- 效果：可承受高达300°C温度
- 范围：全身
- 代价：体温永久升高至38-39°C

**主动能力：焰之爆发**
- 效果：手掌喷射火焰，持续5秒
- 威力：可以融化铁
- 消耗：每次破碎1片感知碎片
- 冷却：30分钟
- 限制：会灼伤自己的手

**永久代价**：
- 体温永久升高
- 容易暴躁、易怒
- 在寒冷环境会难受
- 逐渐失去"冷静"的能力

**第2道材料需求**：
- C级焰仆的火核

**第3道材料需求**：
- B级炎魔的地狱核心

**守夜人评价**：

**永久代价**：
- 皮肤开始长出细小鳞片
- 眼睛开始"鱼类化"（侧面视野变宽）
- 开始"渴望"深海

**第2道材料需求**：

**第3道材料需求**：
