## **⟦ Ⅰ. 理智镜（灵魂之镜）⟧**

### **什么是理智镜？**

**定义**：
- 每个人都有一面"理智镜"
- 它不是物理存在
- 它在你的"精神层面"
- 只有特定方法才能"看到"

**如何看到？**

**方法1：守夜人的"镜室"**
- 守夜人基地有特殊房间
- 房间里有一面"真实之镜"（Safe级异常物品）
- 站在镜子前
- 你能看到你的"理智镜"

**方法2：自己能感觉到**
- 闭上眼
- 深呼吸
- 集中精神
- 你能"感觉"到你的镜子
- 能感觉到它的完整度
- 能感觉到裂痕的位置

**方法3：印记持有者的"直觉"**
- 所有有印记的人
- 都能隐约"看到"自己的理智镜
- 在梦中
- 在冥想时
- 在压力巨大时

## **⟦ Ⅱ. 理智碎片 ⟧**

### **什么是理智碎片？**

**定义**：
- 当你的理智镜破裂
- 掉落的"碎片"
- 它们是你"人性"的一部分

**碎片的数量**：
- 一面完整的理智镜
- 由**100片碎片**组成
- 每片碎片 = 1%理智

**碎片的外观**：
- 像镜子碎片
- 半透明
- 发出微弱白光
- 每片都有独特的形状

---

### **碎片的分类**

**理智碎片有不同类型**：

#### **核心碎片（20片）**
- 位于镜子中心
- 最大、最重要
- 代表：
  - 自我认知
  - 核心记忆
  - 人格本质
- **如果核心碎片碎裂 = 人格崩溃**

#### **记忆碎片（40片）**
- 环绕核心
- 中等大小
- 代表：
  - 重要记忆
  - 关系记忆
  - 经历记忆
- **如果记忆碎片碎裂 = 遗忘**

#### **情感碎片（30片）**
- 分布在外围
- 较小
- 代表：
  - 情感反应
  - 共情能力
  - 情绪稳定
- **如果情感碎片碎裂 = 冷漠**

#### **感知碎片（10片）**
- 最外围
- 最小
- 代表：
  - 现实感知
  - 时间感知
  - 空间感知
- **如果感知碎片碎裂 = 幻觉**

---

### **碎片的破碎顺序**

**重要机制**：
> **碎片不是随机破碎的**  
> **它们按照"从外到内"的顺序破碎**

**破碎顺序**：
```
感知碎片（最先）
    ↓
情感碎片
    ↓
记忆碎片
    ↓
核心碎片（最后）
```

**这意味着**：

**早期（失去感知碎片）**：
- 你开始看到幻觉
- 你开始听到声音
- 你分不清现实和梦境
- **但你还知道自己是谁**

**中期（失去情感碎片）**：
- 你变得冷漠
- 你不再关心他人
- 你无法感受情绪
- **但你还记得重要的人**

**后期（失去记忆碎片）**：
- 你开始遗忘
- 你忘记了朋友的脸
- 你忘记了母亲的名字
- **但你还记得"自己"**

**末期（失去核心碎片）**：
- 你忘记了自己是谁
- 你不知道自己为什么而战
- 你不知道自己是人还是怪物
- **你已经不是"你"了**

## **⟦ Ⅲ. 理智损失的具象化表现 ⟧**

**外观变化**：
- 镜子最外圈开始碎裂
- 几片小碎片掉落
- 黑色从边缘渗入

**具体表现**：

**视觉异常**：
- 偶尔看到"影子"在动
- 墙角有"东西"在晃动
- 夜晚看到"轮廓"

**听觉异常**：
- 听到"窃窃私语"
- 听到"脚步声"（没有人）
- 听到"自己的名字"被呼唤

**时间感知异常**：
- 感觉时间变快/变慢
- 不确定"刚才"发生了什么
- 分不清是白天还是夜晚

**空间感知异常**：
- 感觉房间"不对劲"
- 门的位置"好像变了"
- 走廊"好像变长了"

- Ⅰ级：每3个月
- Ⅱ级：每2个月

- Ⅳ级：每周
- Ⅴ级及以上：每天

**过程**：

**步骤1：进入镜室**
```
镜室是一个黑暗的房间
中央有一面巨大的镜子
这面镜子是Safe级异常物品
SCP-████"真实之镜"
它能显示人的"理智镜"
```

```
你站在镜子前

放松

```

```
镜子里出现的不是你的身体

医生会记录：
- 总体完整度
- 裂纹分布
- 碎片数量
```

## **⟦ Ⅴ. 理智恢复的具象化 ⟧**

### **方法1：自然恢复（粘合碎片）**

**机制**：
- 远离异常
- 充分休息
- 正常生活
- 镜子会"自我修复"

**具象化表现**：
```
你闭上眼
观察你的理智镜
你能看到：

细小的裂纹在"愈合"
像伤口在结痂
很慢，但在发生

掉落的小碎片在"回归"
它们慢慢飞回镜子
重新粘合

白光在增强
黑色在减退
```

**速度**：
- 每周约恢复1-2片碎片（1-2%）
- 只能恢复"掉落不久"的碎片
- 掉落太久的碎片无法自然恢复

---

### **方法2：药物恢复（强制粘合）**

**机制**：
- 服用"清醒药剂"
- 药物强制"粘合"碎片
- 快速但不稳定

**具象化表现**：
```
你喝下药剂
感觉一股"力量"涌入精神

你能看到理智镜：
碎片被"强行"拉回
用某种"胶水"粘合
快速、粗暴

但粘合不完美
还能看到裂痕
还能看到接缝

白光增强了
黑色减退了
但不够"自然"
```

**效果**：
- 立即恢复3-5片碎片（3-5%）
- 但碎片不牢固
- 容易再次破碎

**副作用**：
- 头痛（大脑在适应）
- 恶心（精神在排斥）
- 依赖（需要更多药物维持）

---

### **方法3：仪式恢复（替换碎片）**

**机制**：
- 使用"记忆恢复仪式"
- 用"记忆碎片"替换"其他碎片"
- 本质：牺牲记忆换取核心稳定

**具象化表现**：
```
你躺在仪式台上
SCP-████"记忆之书"在你面前打开

你能看到：
书在"读取"你的记忆碎片
它选中了几片记忆碎片
它们发出金色的光

然后...
这些记忆碎片被"提取"出来
被书"吸收"了
消失了

但作为交换：
其他类型的碎片（情感、感知、核心）
被"修复"了
被"强化"了

你的镜子更稳定了
核心碎片被加固了
白光更亮了

但...
你忘记了一些事
重要的事
```

**具体过程**：

**仪式前**：
```
理智：45%
- 核心碎片：15/20（5片破碎，濒临崩溃）
- 记忆碎片：28/40（12片破碎）
- 情感碎片：20/30（10片破碎）
- 感知碎片：7/10（3片破碎）
```

**仪式中**：
```
"记忆之书"选择了：
- 5片记忆碎片作为"材料"

这5片记忆包含：
- 母亲的脸（童年记忆）
- 初恋的名字（青春记忆）
- 某次旅行（经历记忆）
```

**仪式后**：
```
理智：60%（+15%）
- 核心碎片：20/20（完全修复！）
- 记忆碎片：23/40（失去5片）
- 情感碎片：27/30（修复7片）
- 感知碎片：10/10（完全修复！）

但你失去了那5片记忆
永远
```

## **⟦ Ⅶ. 系统特点总结 ⟧**

### **为什么理智碎片比数字好？**

#### **1. 可视化 ✅**
```
数字：你的理智是67%
碎片：你能"看到"你的镜子，看到裂纹，看到掉落的碎片
→ 更直观，更恐怖
```

#### **2. 有类型 ✅**
```
数字：只是减少
碎片：失去不同类型的碎片有不同后果
→ 感知碎片（幻觉）
→ 情感碎片（冷漠）
→ 记忆碎片（遗忘）
→ 核心碎片（人格崩溃）
```

#### **3. 有顺序 ✅**
```
数字：随机减少
碎片：从外到内破碎
→ 你知道接下来会失去什么
→ 你能"预见"自己的崩溃过程
```

#### **4. 可恢复但有代价 ✅**
```
数字：恢复就是加回来
碎片：用记忆碎片换取其他碎片
→ 恢复有代价
→ 你会失去重要的记忆
→ 更悲剧，更克苏鲁
```

#### **5. 剧情潜力巨大 ✅**
```
场景1：主角第一次"看到"自己的理智镜（震撼）
场景2：主角眼睁睁看着核心碎片裂开（绝望）
场景3：主角用记忆换理智，忘记了女儿（悲剧）
场景4：主角的镜子完全破碎（异化）
```

### **核心象征**

**镜子 = 自我**
- 完整的镜子 = 完整的人
- 破碎的镜子 = 破碎的人
- 黑化的镜子 = 异化的人

**碎片 = 人性**
- 每一片都很重要
- 失去一片就少一点人性
- 全部失去 = 不再是人

**裂纹 = 伤痕**
- 永远不会消失
- 提醒你付出的代价
- 越多裂纹 = 越接近崩溃

**黑色 = 污染**
- 从边缘渗入
- 逐渐吞噬镜子
- 最终取代白光
